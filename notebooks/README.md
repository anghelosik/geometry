# Notebooks

This folder is reserved for Jupyter notebooks exploring the geometry project
interactively.

## Setup

```bash
pip install jupyter
jupyter notebook
```

## Suggested notebooks

| Notebook | Purpose |
|---|---|
| `01_sampling_visualisation.ipynb` | Plot the 9 distributions on the square perimeter |
| `02_distance_histogram.ipynb` | Plot the distance histogram for each method |
| `03_convergence.ipynb` | Show how P(d > L) converges as iterations increase |

## Note

Notebook files (`.ipynb`) and checkpoint folders (`.ipynb_checkpoints/`) are
excluded from version control via `.gitignore`.
